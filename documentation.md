### Setup

- Add `nuxt-vuikit` to `modules` section of your `nuxt.config.js`

```js
{
	modules: [
		'nuxt-vuikit'
	],
	vuikit: {
		defaultTheme: true,
		icons: true
	}
}
```

### Module options

#### `defaultTheme`
Load UiKit3 default theme
- Default: `true`
- Type: `Boolean`

#### `icons`
Load UiKit3 icons
- Default: `true`
- Type: `Boolean`
